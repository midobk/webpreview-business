# SHOWCASE_RULES.md

## Eligibility
- Visually impressive prototype
- Safe to display (no real private info)
- Dashboard-approved by user

## Anonymization Rules
- Replace real business name with generic (e.g., "Modern Cleaning Service")
- Replace real phone/email/address with placeholders
- Remove real logos unless approved
- Use AI-generated/stock images only (no customer photos)
- Label as "Sample/demo concept"

## Showcase Cards
Display as category cards:
- "Modern Cleaning Service Landing Page"
- "Local Contractor Website Concept"
- "Barber Shop Landing Page Preview"
- "Restaurant One-Page Website Concept"
- "Private School Landing Page Concept"

## Layout
- Grid of cards with thumbnail screenshot
- Click opens full anonymized prototype
- No real business names, contact info, or identifying details
